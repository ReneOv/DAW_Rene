  1. Internet: La red más grande, donde existen absolutamente todos los intercambios de información y peticiones.
    World Wide Web: La parte pública del Internet que se navega a trvés de páginas web.
  2. GET: Solicita un recurso específico, solo debe recuperar datos.
    HEAD: Únicamente pide el header de una respeusta.
    POST: Envía cierto recurso a una entidad, causando cambios en el servidor.
    PUT: Reemplaza todas las presentaciones de cierto recurso en el servidor. Se envía junto con la petición.
    DELETE: Borra el recurso especificado del servidor.
    PATCH: Modifica parcialmente un recurso en el servidor. [3]
  3. Se utilizaría el método de POST, ya que es el método encargado de enviar recursos y realizar cambios al servidor.
  4. Se tendría que usar el método de GET.
  5. El código 200 significa que la petición se realizó exitosamente. Por ende, no ocurrió ningún error.
  6. El código 404 significa que el recurso indicado no existe o no está disponible. Es un error del cliente, entonces no es responsabilidad del desarrollador corregirlo.
  7. El código 500 significa que el servidor encontró una condición inesperada y no puede completar la petición del usuario. Éste sí es responsabilidad del desarrollador.
  8. Un elemento depreciado es el elemento que ya no forma parte de la especificación del lenguaje. Por ejemplo, 'menu' y 'applet'. También 'font', que ahora se modifica desde el archivo de CSS.
  9. La revisión de HTML4 a HTML5 añadió y actualizó varias cosas. Tiene un sintaxis más claro, y fue simplificado. HTML5 ahora tiene tags para audio y video. Igual tiene servicios de geolocalización. JavaScript se corre en un thread aparte. Pero, al ser más nuevo, tiene menos compatibilidad con navegadores más antiguos.
  10. Las tablas, en cuanto estructura, tienen celdas, columnas, y bordes. Se pueden fusionar celdas, el alineamiento, y, claro, la cantidad. En cuanto a estilo, se le puede cambiar el fondo, color del borde, y formato del texto dentro de las celdas; Table Header es un tag que inmediatamente hace que se ubique hasta arriba, cambie el estilo de la fuente, y que las demás celdas se ajusten a ese tamaño.
  11. Los controles principales de una forma son: botones, textareas, checkboxes, password, y text. [12]
  12. El navegador que uso (Chrome Dev 68.0.3440.106, Windows 10) tiene una puntuación de 515/555 de compatibilidad con HTML5. [13]
  13. El Ciclo de Vida de los Sistemas Informáticos se refiere al proceso que empieza con la detección de las necesidades y termina con la obsolencia y el reemplazo.
    El Ciclo de Desarrollo de los Sistemas Informáticos comienza cuando se establecen los requisitos, la creación de los modelos del sistema, su implementación, las pruebas y al final el despliegue del sistema.
