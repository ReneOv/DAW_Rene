Investigación
URL Uniform Resource Localizer. Es una referencia a un recurso web que especifica su dirección en Internet. Existe un URL único para cada página y documento de la WWW. [1]
MIME Multipurpose Internet Mail Extensions. Son una serie de especificaciones para el intercambio de archivos a través del Internet. [2]
HTTP HyperText Transfer Protocol. Es el protocolo de comunicación que permite transmitir información en la WWW. [3]
XML eXtensible Markup Language. Es un lenguaje que utiliza marcas para almacenar datos en forma legible. [4]
XHTML eXtensible HyperText Markup Language. Es básicamente HTML, pero expresado como XML correcto. Es más estricto a nivel técnico, pero facilita la búsqueda de errores de esta manera. [5]
HTML5 HyperText Markup Language, versión 5. Es la revisión más moderna del lenjuage básico, HTML. [6]
JS JavaScript. Es un lenguaje de programación interpretado. Se utiliza para sus funciones que son 'client-side'. [7]
PHP PHP HyperText Preprocessor. Es un lenguaje de programación para funciones que son 'server-side'. [8]
JSP JavaServer Pages. Es una herramienta utilizada por desarrolladores para crear páginas web dinámicas. Es similar a PHP, pero utilizando Java. [9]
AJAX Asynchronous JavaScript and XML. Es una técnica para el desarrollo web utilizado para crear aplicaciones interactivas. [10]
Métodos y Errores de HTTP GET: Solicita un recurso específico, solo debe recuperar datos.
HEAD: Únicamente pide el header de una respeusta.
POST: Envía cierto recurso a una entidad, causando cambios en el servidor.
PUT: Reemplaza todas las presentaciones de cierto recurso en el servidor. Se envía junto con la petición.
DELETE: Borra el recurso especificado del servidor.
PATCH: Modifica parcialmente un recurso en el servidor. [3]
1xx: Respuestas informativas. Petición recibida, continuando proceso.
2xx: Peticiones correctas. La petición fue recibida correctamente, entendida y aceptada.
3xx: Redirecciones. El cliente tiene que tomar una acción adicional para completar la petición.
4xx: Errores del cliente. La solicitud contiene sintaxis incorrecta o no puede procesarse.
5xx: Errores del servidor. El servidor falló al completar una solicitud aparenemente válida. [11]
