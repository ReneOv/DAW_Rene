Cuestionario BD vs. DBMS

Qué es una base de datos?
Una base de datos es un conjunto de archivos que está agrupada y estructurada. Cada base está compuesta de tablas que guardan conjuntos de datos.

En qué casos es conveniente usar bases de datos?
Es útil usar bases de datos cuando se necesita guardar una gran cantidad de información que está relacionada.
Al hacer buen uso de las bases de datos se pueden buscar datos rápidamente, reorganizar grupos, y seleccionar tablas relevantes con mucha eficacia.

Qué es un sistema de gestión de bases de datos?
Un DBMS es un sistema que funciona como interfáz entre las bases de datos y el usuario. También funciona para la comuicación entre la base y otros programas.

Enlista las funciones/responsabilidades que tienen los DBMS.

  • Administrar los datos
  • Copias de seguridad y restauración de las mismas
  • Monitoreo para detección de problemas
  • Registro de actividades

Ejemplifica y justifica en qué proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
Para mi proyecto de Fundamentos de Diseño Interactivo, creamos una página que permitía a un estudiante aprender a usar HTML. Como meta de largo alcance, queríamos implementar usuarios para guardar el progreso de cada usuario. Si se hubiera implementado ese sistema, una base de datos para el manejo de toda la información del usuario hubiera sido una excelente elección.
