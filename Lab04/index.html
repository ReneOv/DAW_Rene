Inicio -- Problemas JS

Compara y contrasta Java con JavaScript:

"Java is to JavaScript what car is to carpet"
Java es un lenguaje de porgramación orientado a objetos, mientras que JavaScript está orientado a prototipos. JS utiliza un sintáxis diferente, por ejemplo 'prompt' y 'let'.

Qué métodos tiene el objeto Date?

  • getDate()
  • getDay()
  • getHour()
  • getTime()
  • now()

Qué métodos tienen los arreglos?

  • pop()
  • push()
  • shift()
  • delete()
  • indexOf()

Cómo se declara una variable con alcance local dentro de una función?

Se usa "let".

Qué implicaciones tiene utilizar variables globales dentro de funciones?

Se puede utilizar, cambiar, leer, modificar por todas las otras funciones. Esto significa que podría ser modificada antes de su uso esperado dentro de una función.

Qué método de String se puede utilizar para buscar patrones con expresiones regulares? Para qué podrías utilizar esto en una aplicación web?

Se usa el método de RegExp(), y se puede utilizar para implementar autómatas en tu aplicación web.[1]
